A snake fork written in C++ utilising Direct2D API.


# Obtaining

The Windows (7 and onwards) binaries can be downloaded [here](https://github.com/makuke1234/SnakeD2D/releases).


# Controls

| Key    | Function          |
|--------|-------------------|
| Left   | Move left         |
| Up     | Move up           |
| Right  | Move right        |
| Down   | Move down         |
| Escape | Pause/unpause     |
| F11    | Toggle fullscreen |
